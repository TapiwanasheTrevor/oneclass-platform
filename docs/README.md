# OneClass Documentation

This directory intentionally has a small maintained surface.

## Source Of Truth

The authoritative architecture, remediation status, and expansion policy live in:

- `docs/PLATFORM_ALIGNMENT_REVIEW_AND_ROADMAP.md`

That document is the maintained source of truth for:

- current platform architecture
- verified readiness state
- guarded or experimental areas
- expansion rules and completion sequencing

## Reference Material

The PDF below is retained only as historical/reference input:

- `docs/OneClass_Platform_Reference_v2.docx.pdf`

It is not the runtime source of truth and must not override implemented contracts.

## Repo Rule

Do not add parallel planning, status, or architecture documents back into `docs/`.
If the platform state changes, update `docs/PLATFORM_ALIGNMENT_REVIEW_AND_ROADMAP.md`.
